### 环境
```sh
pip install nvidia-cufile-cu12

2. 安装 Python 绑定
有了底层的 libcufile.so，您还需要它的 Python 接口（绑定）。根据您使用的 CUDA 版本，尝试安装对应的 Python 包。例如，对于 CUDA 12.x，可以尝试：
pip install nvidia-cufile-cu12
请注意，具体的包名和可用性可能因 CUDA 主版本号而异。您可以搜索 nvidia-cufile-cu*来找到适合您 CUDA 版本的包。

```
